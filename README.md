# Marquinho Eletricista

Site institucional desenvolvido para apresentar os serviços do Marquinho Eletricista, com foco em divulgação profissional, geração de contato e conversão pelo WhatsApp.

O projeto foi pensado para transmitir confiança, organização e clareza, destacando os principais serviços elétricos, diferenciais de atendimento, portfólio de trabalhos já realizados, avaliações de clientes, área atendida e perguntas frequentes.

## Objetivo

Criar uma landing page moderna, responsiva e direta, facilitando o contato rápido com clientes de Visconde do Rio Branco e região.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts

## Estrutura do projeto

- `index.html`: estrutura principal do site
- `css/style.css`: estilos visuais e responsividade
- `javascript/script.js`: interações do menu, animações e abas do portfólio
- `assets/`: imagens, vídeos e arquivos visuais utilizados no site

## Destaques do site

- Layout moderno com foco em apresentação profissional
- Versão responsiva para desktop e mobile
- Botões de ação para orçamento no WhatsApp e ligação direta
- Seções organizadas para serviços, diferenciais, avaliações e FAQ
- Portfólio com fotos e vídeos de serviços realizados

## Finalidade

Este repositório serve como base do site de divulgação profissional do Marquinho Eletricista, podendo ser utilizado para publicação, manutenção e futuras melhorias do projeto.
